chordCloud

Zonar is a full stack application inspired by the Discord messaging app. It can be used to live chat with friends via direct messaging or user created servers. This application uses Javascript, HTML, CSS, Node.js, React, Redux, Python, Postgres, Flask, Websocket, and Render technologies.
